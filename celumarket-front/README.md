# 📱 CeluMarket Frontend — Astro & Tailwind v4

Esta es la aplicación web cliente de CeluMarket, construida con **Astro**, **TypeScript** y **Tailwind CSS v4** con empaquetador Vite.

## 🚀 Comandos Rápidos

Asegúrate de estar en el directorio `celumarket-front` antes de correr estos comandos:

| Comando | Acción |
| :--- | :--- |
| `pnpm install` | Instala todas las dependencias del frontend. |
| `pnpm dev` | Inicia el servidor local de desarrollo en `http://localhost:4321`. |
| `pnpm build` | Compila la página estática para producción (se genera en la carpeta `dist/`). |
| `pnpm preview` | Previsualiza localmente el build de producción antes de subirlo. |

## 🔗 Configuración de API
El frontend se conecta con la API de .NET. Recuerda crear tu archivo `.env` en la raíz de esta carpeta:
```env
PUBLIC_API_URL=http://localhost:5220/api
```
